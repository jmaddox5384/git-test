VS Code Live Server Extension Setup

How to Install and Use VS Code's Live Server Extension.

Video Install | Use

Installation Instructions

  1. In Visual Studio Code, click the Extension icon in the left column toolbar. Extensions icon in the left column toolbar.
  2. In the search bar that apperars, type Live Server.
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for live Server that appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server
  • In VS Code's Explorer, right click (windows) or control+click (macOS) on HTML file. From the context menu that apperars, select open with live Server.
